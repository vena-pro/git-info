Клонирование проекта в папку  nameProject:
git clone htts://... nameProject

Добавить файлы коммит:
git add .

Записать коммит:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull

Авторство коммитов:
git config --global user.email "Ваш емацл"
git config --global user.name "Ваше имя"
